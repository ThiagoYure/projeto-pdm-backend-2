### Executando
Instale as dependências usando
```
yarn
```

Ininicie o metro
```
yarn start
```

Compile o código fonte
```
npx react-native run-android 
```

Para mais detalhes acesse o tutorial do próprio react native [aqui](https://reactnative.dev/docs/environment-setup).
